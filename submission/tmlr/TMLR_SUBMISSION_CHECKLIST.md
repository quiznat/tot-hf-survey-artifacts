# TMLR Submission Checklist

## Pre-Submission Integrity
- [x] Human-only byline in canonical manuscript.
- [x] Agent/system contribution explicitly separated from byline authorship.
- [x] High-stakes claims mapped to evidence artifacts.
- [x] Scope and claim-boundary language calibrated to cited evidence.

## Double-Blind Package
- [x] Anonymous manuscript generated (`submission/tmlr/anonymous/paper-anonymous.html`).
- [x] Anonymous LaTeX generated (`submission/tmlr/anonymous/main.tex`).
- [ ] Anonymous PDF compiled and reviewed for residual identifiers.
- [ ] Manual scan completed for direct identifiers in figures, appendix text, and links.
- [ ] Final submission PDF compiled using the official TMLR stylefile/template.

## OpenReview Metadata
- [ ] OpenReview profile is active for submission.
- [x] Account setup reviewed (`submission/tmlr/OPENREVIEW_ACCOUNT_SETUP.md`).
- [ ] Title and abstract copied into OpenReview form.
- [ ] Keywords entered.
- [ ] Primary field and subfields selected.
- [ ] Conflicts of interest entered.
- [x] Suggested reviewers entered (optional but recommended).

## Policy and Disclosure
- [ ] AI/system-assistance disclosure finalized for camera-ready stage.
- [ ] Data/material availability statement finalized.
- [ ] Citation and originality checks completed.

## Final Submit Gate
- [ ] Upload anonymous PDF to OpenReview.
- [ ] Upload supplementary artifacts only if they do not break anonymization.
- [ ] Confirm submission receipt and revision window dates.
